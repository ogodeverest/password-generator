## Run Locally

Clone the project

```bash
  git clone https://github.com/ogodeverest/password-generator.git
```

Go to the project directory

```bash
  cd password-generator
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```
